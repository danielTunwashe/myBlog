About

This Blogging Application helps different authors/users to create different post about various events happening. The users are authenticated and also properly verified and also the application makes sure that it only a specified author who creates a post can edit or delete the post. Also, this application has a special feature called the Admin Panel. The Admin Panel helps an Admistrator to monitor all the affairs of the application and perform only Admin Functionalities.
